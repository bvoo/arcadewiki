## Overview

The Smash Box is a leverless controller released in 2017 by Hit Box Arcade. The retail price was $249.99 USD. The Smash Box is no longer available to purchase.

## Development

### Release History

The Smash Box was released in 2017 by Hit Box Arcade.

The Smash Box uses a steel case construction with an acrylic top panel. 

## Technical Specifications

### Materials

**Case**: Steel with acrylic top panel.

**Buttons**: Sanwa OBSF-24 arcade buttons (24mm diameter). 

**Compatibility**: The 24mm snap-in buttons mount in metal or wooden control panels up to 3.7mm thick using 24mm mounting holes.

### Button Configuration

The Smash Box uses Sanwa OBSF-24 buttons throughout.

**Stock Switch**: Sanwa SW-68 microswitch

### Layout

The Smash Box uses a 19-button layout for FGC (Fighting Game Community) compatibility. The layout includes:

- Directional buttons
- Action buttons
- Modifier buttons
- Profile switch toggle

### USB Interface

**Port**: USB-B (detachable cable system).

**Cable**: Detachable GameCube cable included.

**Compatibility**: Native GameCube and Wii compatibility. Wii U, Switch, and PC compatibility via adapter.

## Software

### Smash Box Designer

Hit Box Arcade provides the Smash Box Designer software for Windows. The software enables:

- Button remapping across three profiles
- Tilt modifier value adjustment
- SOCD (Simultaneously Opposing Cardinal Directions) cleaning configuration
- Analog angle customization
- Layout import and export
- Nunchuk configuration for analog stick emulation

### Built-in Profiles

The controller ships with three pre-configured profiles:

- Default: Ultimate
- Profile 2: Melee
- Profile 3: Rivals of Aether

Profile switching is done via hardware toggle without software connection.

## Retail Package

The Smash Box was sold with the controller, detachable GameCube cable, and USB adapter for PC/Switch compatibility.

### Optional Accessories

Available from Hit Box Arcade: Clear acrylic cover replacements, carrying cases, and Smash Box shirts.
