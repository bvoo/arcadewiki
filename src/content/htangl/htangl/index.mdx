## Overview

The HTangl is an open-source, rectangular, all-digital controller.

The HTangl is similar in operation to the B0XX and Frame1 but uses a unique "top row home row" layout that moves the right-hand button rows further down toward the C-Stick cluster. This allows the right hand to rest on the top row of buttons rather than the bottom, making it easier to switch between rows by curling fingers inward rather than moving the whole hand.

## Development

The HTangl project was created as an open-source alternative to commercial rectangle controllers. It builds on designs from the [B0XX](../b0xx/b0xx), [Frame1](../frame1/frame1-light), and GCCMX, and runs a modified version of the HayBox firmware.

HTangl also developed several companion DIY projects:

- **DIDIY** - an all-in-one PCB combining all components needed for a rectangle controller onto a single board, featuring a Raspberry Pi Pico.
- **MX1S** - small PCBs that attach to any standard MX switch to enable solderless wiring, inspired by Crane's MX1 boards.
- **BudgeTangl** - a budget-oriented laser-cut case design using the HTangl layout, with all panels orderable from SendCutSend.
